Principais Formatações

Negrito/ Destaque

use a tag <b> </b> e tag <strong> </strong> (strong esta na semântica);

Italico <i> </i> e tag <em> </em> (em esta na semântica);

Marca texto <mark> </mark>;

Texto Grande <big> </big> (Tag em desuso);

Texto Pequeno <small> </small>;

Texto exculido <del> </del>;

Texto Inserido <ins> </ins> e <u> </u>(ins esta na semântica);

Texto sobreescrito <sup> </sup> ;

Espaçamento para codigos <code> </code> e <pre> </pre>;

Exemplo de codigo:


        void main()
        {
        while (1)
        {
        output_bit(PINOLED, TRUE); // Liga o Led
        delay_ms(5000); // Aguarda 3 segundos
        output_bit(PINOLED, FALSE); // Desliga o Led
        delay_ms(2000); // Aguarda 2 segundos
        }
        }

Citações:

aspas: <q> </q>

use <blockquote> </blockquote> para citar termos e seus significados e direitos autorais

use <abbr> </abbr> para citar termos e seus significados e direitos autorais
